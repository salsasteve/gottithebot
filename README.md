# Gotti The Bot

This is a simple chatbot that uses the ChatGPT library to generate responses to messages received from users on Telegram.

## Prerequisite

- docker
- docker-compose
- .env

## Usage

```
docker-compose build
docker-compose --env-file ./env up
```
